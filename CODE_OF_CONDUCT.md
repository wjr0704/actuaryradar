# Code of Conduct

## Our Pledge

We pledge to make participation in this project a respectful and inclusive experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Welcome different levels of experience.
- Focus feedback on the work, not the person.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination.
- Personal attacks.
- Publishing private information.
- Repeated disruptive behavior.

## Enforcement

Project maintainers may remove comments, close issues, or restrict participation when behavior is inconsistent with this Code of Conduct.
